# スレッド

最近のOSの大半は、コードの実行を一つのプロセスで実行しており、OSは一度に複数のプロセスを管理する。
あなたは自身のコードを、同時実行可能な独立した部分に分けることができる。
これらの独立実行する部分はスレッドと呼ばれている。
