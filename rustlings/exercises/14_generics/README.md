# ジェネリクス

ジェネリクスは型を生成や様々なケースに対応できる機能のトピックです。
ジェネリクスは色んな方法でコードの重複を減らすことができとても便利ですが、かなり複雑な構文が必要になることがある。
すなわち、ジェネリックであるためには、ジェネリック型が実際にどの型に対して有効であるとみなされるかを、細心の注意を払って指定する必要がある。
最も簡単で一般的な使い方は型のパラメータです。
